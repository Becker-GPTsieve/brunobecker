Gerador de Números Primos baseado no Crivo Becker–GPT
======================================================

Descrição:
-----------
Este programa em Python puro gera números primos até um limite informado pelo usuário,
utilizando um algoritmo inspirado no Crivo Becker–GPT. Não utiliza bibliotecas externas
e roda diretamente no console.

Funcionalidades:
-----------------
- Interface simples em console.
- Solicita ao usuário o número limite.
- Calcula e exibe o tempo total de execução.
- Gera um arquivo de saída "primos_becker_gpt.txt" com:
    * Tempo de execução
    * Total de primos gerados
    * Lista dos primos separados por vírgula
- Mantém o console aberto após a execução para leitura dos resultados.

Uso:
-----
1. Execute o arquivo `gerador_primos_becker_gpt.py` com Python 3.
2. Insira o número limite quando solicitado.
3. Verifique o arquivo gerado com os resultados.
